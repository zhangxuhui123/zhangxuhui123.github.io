数据库基础操作
发布于 2020-04-07   更新于 2020-04-15

注意

  1. SQL语句以分号（;）结尾，所以可以使用空格和缩进来增加可读性。
  2. MySQL数据库的SQL语句不区分大小写，为了显眼，关键字建议使用大写。
  3. 3种注释：
    • 单行注释： – 注释内容（–空格） 或 # 注释内容（MySQL特有）（#可以加空格也可以不加）
    • 多行注释： /注释内容/

SQL分类

  1. DQL (Data Query Language)数据查询语言。select，where，order by，gruop by和having等。

    * 查询表中的数据

  2. DML（Data Manipulation Language）数据操作语言。insert，update和delete等

    * 增删改表中的数据

  3. DCL （Data Control Language）数据控制语言。

    * 授权的（了解）

  4. DDL （Data Definition Language）数据定义语言。alter、create和drop 等

    * 操作数据库和表的

    DDL ：操作数据库，表

    1. 操作数据库：CRUD(增删改查)

      1. C(create):创建。
    * 创建一个数据库：create database 数据库名称；
    * 创建一个不存在的数据库（如果存在则不创建新的）（避免创建一个已经存在的数据库报错）： create database if not exists 数据库名称；
    * 创建一个指定字符集的数据库： create database 数据库名称  character  set  字符集名称；
    * 练习：创建db4 数据库，判断是否存在并致顶字符集为gbk： create database if not exists  db4 character set gbk；
2. R(Retrieve):查询。
    * 查询所有数据库的名称：show databases;
    * 查看某个数据据库的字符集：查看某个数据库的创建语句：show create database 数据库名称；
3. U(Update):修改。
    * 修改数据库的字符集： alter database 数据库名称 character set 字符集名称；
4. D(Delete)：删除。
    * 删除数据库： drop database 数据库名称；
    * 删除一个存在的数据库（避免删除一个不存在的数据库报错）： drop database if exists 数据库名称；
5. 使用数据库：进入数据库。
    * 查询当前正在使用的数据库名称： select database()；
    * 使用数据库： use 数据库名称；（不用再加database了）

      1. 操作表

        1. C(create):创建。
    * create table 表名（
        列名1 数据类型1，
        列名2 数据类型2，
        ...，
        列名n 数据类型n）;
    * 注意 
        1. 最后一列不需要加逗号。
        2. 数据库中常用数据类型
            1. int ：整数类型。
                * 例  age int，
            2. double： 小数类型
                * score double（数据长度，小数点后位数），
                * 例  score double（5，2），
            3. date：日期类型，只包含年月日，yyyy-MM-dd
            4. datetime：日期类型，包含年月日时分秒，yyyy-MM-dd HH:mm：ss
            5. timestamp：时间戳类型，包含年月日时分秒，yyyy-MM-dd HH:mm：ss
                * 若将来不给这个字段赋值，或赋值为null则默认使用当前系统时间来自动赋值。
            6. varchar：字符串类型
                * name varchar（最大字符长度），
                * 例 name varchar（20），
                * zhangsan 8个字符； 张三 两个字符。

2. R(Retrieve):查询。
    * 查询某个数据库中所有表的名称：show tables；
    * 查询表结构： desc 表明；
3. U(Update):修改。
4. D(Delete)：删除。
